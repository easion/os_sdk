.::Netbas OS:.


A new kind of desktop operating system.

Netbas OS (the old name Jicama) is an entirely different desktop operating system.plans to make a run on multiple platforms entertainment-oriented applications, This makes Netbas a general purpose operating system. Including the mainstream embedded systems and PC-(WinCE Like), the current can be part of a binary-compatible linux systems, and developed their own proprietary GUI Window System: GI.
GI Windows system based on kernel mode, the user layer simulation libX11, and developed their own widget toolkit. Will be able to run SDL, FLTK-1, Mesa, Doom, links web browser, VNC.
Netbas current development is still in alpha stage, the release of an experimental version.
